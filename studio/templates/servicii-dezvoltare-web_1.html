Dezvoltare web

Leviatan Web >> Servicii >> Dezvoltare web

Oferta noastra de servicii de dezvoltare web include patru pachete:

     WEB STANDARD - pachet destinat companiilor care doresc doar un site static, de prezentare. El include si serviciul de optimizare SEO on-page.
     WEB MEDIU - pachet destinat companiilor care doresc mai mult decat un site de prezentare. Puteti prezenta clientilor mai multe informatii, pagini secundare, mai multe imagini, sau puteti avea o galerie foto sau video pentru descrierea produselor dumneavoastra. Pentru acest pachet bannerele sau meniurile pot fi animate.
     WEB BUSINESS - include toate serviciile din cadrul pachetului WEB MEDIU, dar pe langa acestea, mai sunt oferite: formular de inscriere pentru newsletter, motor de cautare intern, modul de administrare (CMS). Modulul de administrare va ofera posibilitatea de a modifica oricand continutul site-ului, fara a fi nevoie de ajutorul unui programator web.
     WEB PORTAL - se adreseaza companiilor care doresc, pe langa serviciile incluse in pachetul WEB BUSINESS, prezentarea si vanzarea online a produselor lor.

     Pentru a vedea etapele de dezvoltare pentru proiectul dumneavoastra, va rugam sa mergeti la pagina Dezvoltare web - modalitate de lucru

NOTA:

• estimarea pretului si a duratei de dezvoltare se face in functie de complexitatea site-ului (numarul de imagini optimizate, numarul de categorii / subcategorii si produse introduse initial, numarul de pagini secundare, servicii etc.); • costul final al site-ului va fi stabilit in urma discutiilor cu dumneavoastra;
• pachetele noastre, prezentate mai sus, respectiv serviciile incluse in acestea, pot fi modificate in functie de cerintele dumneavoastra;
• durata de dezvoltare a site-ului va fi calculata din momentul primirii materialelor de la dumneavoastra (text, imagini, video etc.).
